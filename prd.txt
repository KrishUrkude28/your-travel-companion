PART 1 — PRODUCT REQUIREMENTS DOCUMENT (PRD)
1. Product Overview
TravelSathi is an AI-powered travel booking web application built for the Indian market. It provides a one-stop platform for discovering, planning, and booking travel — combining the core booking functionality of platforms like MakeMyTrip with two unique differentiators: an AI trip planner powered by Gemini API and a local guide booking marketplace.

3. Row Level Security (RLS) Policies
Critical Supabase Security
Enable RLS on ALL tables. Without this, any authenticated user can read/write all data.
Run these policies in Supabase SQL editor after creating tables.
Table
Policy
Rule
profiles
SELECT / UPDATE
auth.uid() = id (users see only their profile)
guides
SELECT
Public read — anyone can browse guides
guides
INSERT / UPDATE
auth.role() = 'admin' (admin-only writes)
bookings
SELECT
auth.uid() = user_id
bookings
INSERT
auth.uid() = user_id AND status = 'pending'
itineraries
ALL
auth.uid() = user_id
reviews
SELECT
Public read
reviews
INSERT
auth.uid() = user_id
payments
SELECT
auth.uid() = (SELECT user_id FROM bookings WHERE id = booking_id)
4. Gemini API Integration
Edge Function: generate-itinerary
// supabase/functions/generate-itinerary/index.ts
import { serve } from 'https://deno.land/std/http/server.ts'
serve(async (req) => {
  const { destination, days, budget, interests, count } = await req.json()
  const GEMINI_KEY = Deno.env.get('GEMINI_API_KEY')
  const prompt = `Create a ${days}-day itinerary for ${destination}.
    Budget: INR ${budget}. Travellers: ${count}. Interests: ${interests.join(', ')}.
    Return ONLY valid JSON: { title, summary, budget_breakdown, days: [...] }`
  const res = await fetch(`https://generativelanguage.googleapis.com/v1beta/models/
    gemini-1.5-flash:generateContent?key=${GEMINI_KEY}`, {
    method: 'POST', body: JSON.stringify({ contents: [{ parts: [{ text: prompt }] }] })
  })
  const data = await res.json()
  const text = data.candidates[0].content.parts[0].text
  const itinerary = JSON.parse(text.replace(/```json|```/g, '').trim())
  return new Response(JSON.stringify(itinerary), { headers: { 'Content-Type': 'application/json' } })
})
Environment Variable
Add GEMINI_API_KEY in Supabase Dashboard → Settings → Edge Functions → Secrets
Get your free Gemini API key at: https://aistudio.google.com/app/apikey
5. Razorpay Integration
Frontend Flow
User clicks 'Pay Now' on order summary page
Call Supabase Edge Function create-order → returns razorpay_order_id
Load Razorpay script: https://checkout.razorpay.com/v1/checkout.js
Open Razorpay modal with: key, amount, currency, order_id, name, description
On payment success: handler receives payment_id, order_id, signature
Call Supabase Edge Function verify-payment → stores payment record
Update bookings table: status = 'confirmed', payment_id stored
Redirect to booking confirmation page
